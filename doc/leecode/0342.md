# 4的幂
[出处](https://leetcode-cn.com/problems/power-of-four/)  
给定一个整数 (32 位有符号整数)，
请编写一个函数来判断它是否是 4 的幂次方。  

示例 1:
```text
输入: 16
输出: true
```

示例 2:
```text
输入: 5
输出: false
``` 

进阶：
你能不使用循环或者递归来完成本题吗？  

## 答案
1. 解法说明 
- 必要条件： 
4 * 4 =（3+1）*（3+1）   
= 3*3 + 2*3*1 + 1*1  
= （3+2）* 3 + 1
同理  。。。。  
4的幂 除以3  余数一定是 1

充要条件： 除以3 余1 并且是偶数,则是4的幂
