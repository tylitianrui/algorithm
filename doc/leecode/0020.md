# 20. 有效的括号

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

- 左括号必须用相同类型的右括号闭合。
- 左括号必须以正确的顺序闭合。  

注意空字符串可被认为是有效字符串。

示例1
```text
输入: "()"
输出: true
```

示例2

```text
输入: "()[]{}"
输出: true

```

示例3

```text
输入: "(]"
输出: false
```

示例4 
```text
输入: "([)]"
输出: false
```
